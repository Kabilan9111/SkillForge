🎤 Mock Interview - Quick Test

Test the Mock Interview feature before integration

Track: Not Set

ℹ️ Browser Requirements

  • Browser: Chrome or Edge (recommended)
  • Microphone: Required for voice recording
  • Connection: HTTPS or localhost
  • Backend: Make sure server is running at http://localhost:5000

📋 Testing Instructions

  1. Click "Setup Test Track & Level" to configure localStorage
  2. Click "Open Mock Interview" to navigate to the interview page
  3. Click "Start Interview" on the interview page
  4. Answer 5 text questions (Phase 1)
  5. After phase transition, answer 5 voice questions (Phase 2)
  6. Review final scores on completion screen

⚠️ Important Notes

  • Grant microphone permission when prompted (required for voice round)
  • Speak clearly during voice recording for accurate transcription
  • If API fails, fallback questions will be used automatically
  • Session persists across refreshes - use "Resume Interview" to continue

🔧 Troubleshooting

  • Validation Error: Click "Setup Test Track & Level" first
  • No Voice Recognition: Check browser/OS microphone permissions
  • No Questions Loading: Backend server may not be running
  • Audio Not Playing: Check system volume and browser audio settings
