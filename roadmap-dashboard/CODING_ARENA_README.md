# DSA Practice Coding Arena

## Overview
A full-screen, professional coding environment integrated into SkillForge dashboard that combines the best features of VS Code and LeetCode for practicing data structures and algorithms.

## Features

### 🎯 **Three-Panel Layout**
- **Left Panel**: Problem context, examples, constraints, and complexity requirements
- **Center Panel**: VS Code-like code editor with syntax highlighting and multi-language support
- **Right Panel**: Three tabs for Test Cases, Execution Output, and AI Mentor

### 💻 **Code Editor**
- **Multi-language support**: Python, JavaScript, Java, C++
- **Smart editing**: Tab indentation, auto-formatting
- **Pre-loaded templates** for each language
- **Code persistence**: Drafts saved automatically to localStorage
- **Line/column tracking**: Professional editor information bar

### 🤖 **AI Coding Mentor**
The AI assistant is designed to **teach, not solve**. It has four distinct modes:

1. **Hint Mode** 🡆
   - Provides progressive hints (3 levels)
   - Guides thinking without revealing solutions
   - Example: "Think about what data structure allows O(1) lookups..."

2. **Debug Mode** 🔍
   - Detects common mistake patterns in code
   - Explains errors in simple terms
   - Example: "You're checking consecutive elements, but you need to check ALL pairs"

3. **Optimize Mode** ⚡
   - Suggests time/space complexity improvements
   - Explains why optimizations work
   - Example: "Use hash map to reduce from O(n²) to O(n)"

4. **Explain Mode** 📚
   - Breaks down DSA patterns and concepts
   - Teaches reusable problem-solving techniques
   - Example: "This uses the Two-Pointer pattern, common for sorted array problems"

### 🧪 **Test Case Execution**
- **Run Code**: Test against visible test cases
- **Submit**: Evaluate against all test cases (including hidden ones)
- **Status indicators**: 
  - ✓ **Accepted** (green)
  - ✗ **Wrong Answer** (red) with expected vs actual output
  - ⚠️ **Runtime Error** (red) with error explanation
  - ⏱️ **Time Limit Exceeded** (orange)
- **Performance stats**: Runtime and memory usage

### 📊 **Progress Tracking**
All coding activity feeds back into the **Skill Gap Analyzer**:
- Tracks attempts per problem
- Logs time spent
- Identifies mistake patterns (e.g., off-by-one errors, recursion issues)
- Updates weak areas based on coding performance
- Links coding mistakes to missing skills

### 🎨 **Design & UX**
- **Dark theme** matching SkillForge aesthetics
- **Full-screen immersive** workspace (no distractions)
- **Smooth animations** and transitions
- **Responsive panels** with drag-to-resize handles
- **Professional feel**: Looks and feels like real interview prep platforms

## Usage

### Opening a Problem
1. Navigate to **DSA Practice Arena** from sidebar
2. Click any problem in the topics grid
3. Full-screen coding arena opens automatically

### Coding Flow
1. **Read problem** in left panel (description, examples, constraints)
2. **Write solution** in center editor
3. **Run tests** to check correctness
4. **Get AI hints** if stuck (Hint Mode → progressive guidance)
5. **Debug errors** with AI (Debug Mode → identifies issues)
6. **Optimize** solution with AI suggestions
7. **Submit** when ready

### Getting Help
- **Quick Actions**: Pre-defined prompts like "Get a Hint", "Explain Error"
- **Custom Questions**: Type any question in AI input field
- **Mode Switching**: Choose appropriate mode (Hint/Debug/Optimize/Explain)

### Exiting Arena
- Click **Back to Problems** button in top navbar
- Your code is **automatically saved** as a draft
- Return anytime to continue from where you left off

## Technical Architecture

### File Structure
```
roadmap-dashboard/
├── coding-arena-styles.css    # Full CSS for three-panel layout, AI UI
├── coding-arena.js            # Core logic, AI mentor, execution engine
├── enhanced-flows.js          # Problem data, integration with practice page
└── index.html                 # HTML structure for arena
```

### Key Components

#### 1. **Problem Management** (`coding-arena.js`)
- Built-in problem database (Two Sum example included)
- Dynamic problem loading from `DSA_TOPICS` (generated by `enhanced-flows.js`)
- Auto-converts practice page problems to arena format

#### 2. **AI Mentor Knowledge Base** (`coding-arena.js`)
- Problem-specific hints (3 progressive levels)
- Common mistake patterns (regex detection)
- Optimization strategies
- DSA concept explanations

#### 3. **Code Execution Simulator** (`coding-arena.js`)
- Simulates test case execution
- Provides realistic output (Accepted/Wrong/Error/TLE)
- Generates performance metrics

#### 4. **Progress Tracking** (`coding-arena.js`)
- Tracks attempts, time, mistakes per problem
- Stores in localStorage (`codingProgress`, `codingDrafts`)
- Feeds data to Skill Gap Analyzer for weak area identification

### Integration Points

#### Practice Page → Coding Arena
```javascript
// enhanced-flows.js
function openProblemModal(problemId) {
    if (window.CodingArena) {
        window.CodingArena.openProblem(problemId);
    }
}
```

#### Coding Arena → Skill Gap Analyzer
```javascript
// coding-arena.js
function trackProgress(outcome) {
    // Stores mistakes, time, attempts
    // Skill Gap Analyzer reads this data to identify weak areas
    localStorage.setItem('codingProgress', ...);
}
```

## Configuration

### Adding New Problems
Edit `coding-arena.js` and add to the `problems` object:

```javascript
const problems = {
    'your-problem-id': {
        id: 'your-problem-id',
        title: '123. Your Problem',
        difficulty: 'medium',
        description: 'Problem statement here...',
        examples: [
            {
                input: 'nums = [1,2,3]',
                output: '6',
                explanation: 'Sum of all elements'
            }
        ],
        constraints: ['1 <= nums.length <= 1000'],
        timeComplexity: 'O(n)',
        spaceComplexity: 'O(1)',
        testCases: [
            { input: [[1,2,3]], expected: 6 }
        ],
        templates: {
            python: '# Python code template',
            javascript: '// JS code template',
            java: '// Java code template',
            cpp: '// C++ code template'
        }
    }
};
```

### Adding AI Knowledge
Edit the `aiKnowledge` object in `coding-arena.js`:

```javascript
const aiKnowledge = {
    'your-problem-id': {
        hints: [
            { level: 1, content: 'First hint...' },
            { level: 2, content: 'Second hint...' },
            { level: 3, content: 'Final hint...' }
        ],
        commonMistakes: [
            {
                pattern: /some-regex/,
                message: 'Explanation of mistake'
            }
        ],
        optimizations: [
            {
                from: 'Brute Force',
                to: 'Optimized Approach',
                improvement: 'O(n²) → O(n)',
                explanation: 'Why this works...'
            }
        ],
        concepts: [
            {
                name: 'Hash Table Pattern',
                description: 'When and how to use hash tables'
            }
        ]
    }
};
```
